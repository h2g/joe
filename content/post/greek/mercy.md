# ( ἐλεῶ ) 

> # 羅馬書 9:15  
> # 因他對摩西說：`我要憐憫誰就憐憫誰`，要恩待誰就恩待誰。      
> # τῷ Μωϋσεῖ γὰρ λέγει· `Ἐλεήσω ὃν ἂν ἐλεῶ`, καὶ οἰκτιρήσω ὃν ἂν οἰκτίρω. 

# 憐憫 ( Ἐλεήσω )  
  -- 未來 主動 直說語氣 第一人稱 單數
# ( ὃν )  
  -- 關係代名詞 直接受格 單數 陽性
# ( ἂν )  
  -- 質詞 表示可能性、無限性，常用在條件句的結果子句中
# 憐憫 ( ἐλεῶ )  
  -- 現在 主動 假設語氣 第一人稱 單數
